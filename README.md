# Book Search Web
응용프로그래밍화면구현 수행평가를 위한 레포지토리입니다.

## 👉 Details
- 사용자가 검색바에 책 이름을 검색
- 검색한 책 이름의 정보(이름, 가격, 작가, 출판사, ISBN 등)
- 최종적으로는 책을 판매하는 판매 사이트에 이동할 수 있게 만듬을 목적

## 👥 Members

| 이름   | 맡은 역할           |
|--------|---------------------|
| 이용혁 | HTML 담당            |

| 김류진 | 웹 디자인 기획, 로고 담당 |
